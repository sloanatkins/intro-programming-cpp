# Introduction to Programming with C++

This repository contains lab reports and coursework documentation from an introductory programming course focused on C++ fundamentals and problem solving.

## Course Context
**ECE 118 – Introduction to Programming**  
This course introduces computing concepts, program design, and software engineering principles using C++. Topics include problem solving, algorithmic thinking, and foundational programming constructs.

## Topics Covered
- Introduction to computing and program design
- C++ language fundamentals
- Problem decomposition and algorithm development
- Basic software engineering principles
- Documentation and technical reporting

## Repository Structure
- `/labs` – PDF lab reports documenting programming exercises and design work

## Notes
This repository primarily contains written lab submissions rather than source code. It reflects early-stage programming coursework with an emphasis on understanding core concepts and structured problem solving.
